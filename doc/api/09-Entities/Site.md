# Site

## CRUD
| CRUD   | Http Verb | Route      | Description                         |
|--------|-----------|------------|-------------------------------------|
| READ   | GET       | /site/{id} | Retrieve a single Site from it's id |
| UPDATE | POST      | /site/{id} | Update a single Site                |
